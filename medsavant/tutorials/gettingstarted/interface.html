Navigating the Interface

Upon logging in, you will see the Summary page. Here, basic information about your project is displayed.

There are three main sections in MedSavant (four for administrators):

  • Project displays summary information about your project
  • Table lists and allows management of database contents
  • Variants supports the filtration, visualization, and analysis of genetic variants
  • Administrator enables administrative management of the database and project

These sections are listed as gray links along the top of the window. Each section has one or more subsections. Subsections for the active section are listed along the left side of the window. We will explore the features of the various sections and subsections in the other tutorials. For now, click around to gain an understanding of how the program is organized.
