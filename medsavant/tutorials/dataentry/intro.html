Overview

After this lab, you will be able to:

  • manage patient data
  • import your own gene / region lists
  • upload variant files (admin only)

Requirements

You should complete this tutorial with the latest version of MedSavant that is compatible with the server you're connecting to. If you don't have your own server, you can perform this tutorial by connecting to our public servers via the Portal.

Troubleshooting

If you are performing this lab during a workshop, feel free to ask the instructor for help. Otherwise, please email us at support@genomesavant.com.
