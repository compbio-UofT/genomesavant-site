Overview

This tutorial guides you through the process of:

  • installing the Infobright database
  • installing and booting the MedSavant server

Requirements

In order to install the MedSavant server, the machine you are installing should meet the following specifications:

Operating System Red Hat Enterprise Linux 5 or 6.1, Debian 5 “Lenny” or 6 “Squeeze”, CentOS 5.4 or 6.1, Ubuntu 8.04 (32-bit only), Fedora 9 (32-bit only)
Memory4 GB minimum, >8GB recommended
Processor1.8GHz minimum, >2.0 GHz recommended
