Loading a Genome

Savant requires that a genome be loaded before tracks.

For the remainder of this tutotial, we will be using the human reference genome hg19. Load this genome as follows:

  • Click File > Load a Genome.
  • Select Published Genome: this option is appropriate when the genome you are working with is found in the dropdown list of published genomes. For some genomes, you can simultaneously open related tracks e.g. a gene track.
  • Select the Human - Feb. 2009 (GRCh37/hg19) from the dropdown (if it's not already selected).
  • Check the Refseq genes checkbox. Click OK.

If there is some genome that you would like to load, but is not listed in the Load Genome dialog, please let us know. You can always alternatively load a genome in FASTA format by clicking File > Load Genome from File.

Loading Tracks

Savant can load data that's stored a various formats, including BAM, VCF, BED, GFF, and WIG. Some of these files may need to be specially formatted, however, so that you can browse them quickly. BAM files do not need to be formatted, although they are required to be sorted (using Samtools sort function) and have an index with extension .bai in the same directory as the file with extension .bam. Upon requesting a file to be loaded through File > Load Track from File..., Savant will either load it if previously formatted or prompt you to format it. For now, let's instead try loading tracks from URL.

Savant can load tracks from web servers, provided the full URL to a previously formatted track. Below is a link to a read alignments file that is hosted on our web servers. Copy the full URL, then load it using File > Load Track from URL.... Paste the URL in the ensuing dialog and click OK.
