Creating a project

In order to start using MedSavant, you'll need to create a database and a project. A database may contain multiple projects. Let's start by creating a database.

Creating a MedSavant Database

Open the MedSavant client. Click the Create Database button. Enter the appropriate connection information, including administrator credentials. Make up an appropriate database name. This can't be changed after creation. Click the Create button when ready.

A database will be created on the server. Be patient, it takes about a minute.

Creating a MedSavant Project

Login using the same connection information used to create the database. Since this is the first time logging into the database, you'll be prompted to create a project. You'll proceed through the project creation wizard.

You will need to provide some information to customize your project:

  • Project Name - a unique name for the project (e.g. "Autism Exome Project")
  • Individuals - what information to store about patients
  • Custom VCF Fields - what information from the INFO field to parse and make searchable
  • Reference - the reference genome variants refer to

Continue through the wizard, clicking Create Project when ready.

After your project has been created, you can login and start using MedSavant. Proceed to the Administration tutorial to learn how to add users and to modify projects.
