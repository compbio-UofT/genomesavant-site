Part 5: Exporting Images from Savant

If you've found an region of interest that you'd like to show in presentation or in a publication, you can export images from Savant choosing File > Export Track Images .... Then you must select the tracks to include in the image. Press the > > button to include all tracks. Press Ok. Finally, specify a file name and press Save.
