Exporting Savant Images

While screenshots are nice, they contain a lot of superfluous content (e.g. buttons and menus) which don't need to be included. Savant has an export option that saves only the relevant track information to an image file for you to use in presentations or publications.

Once you've found an region of interest, you can export it into a succinct image by choosing File > Export Track Images .... Select tracks to include one-by-one by using the > button, or press the > > button to include all tracks. Press Ok. Finally, specify a file name (including the .png extension) and press Save.

Congratulations, you're now finished with the basics!
