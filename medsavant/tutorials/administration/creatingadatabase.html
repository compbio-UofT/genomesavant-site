Creating a Database

In order to start using MedSavant, you'll need to create a database in which projects reside. A database may contain multiple projects. A server may contain multiple databases, too.

Creating and Configuring a Database

Open the MedSavant client. Click the Create Database button. Enter the appropriate connection information, including administrator credentials. Make up an appropriate database name. This can't be changed after creation. Click the Create button when ready.

A database will be created on the server. Be patient, it takes about a minute. Once the database is created, you can login. The first time you do, you'll be prompted to create a project, which is discussed in the next section.
