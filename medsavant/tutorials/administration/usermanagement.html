User Management

MedSavant users are added or removed via the "Users" section of the admin section, using the () or remove () buttons, respectively, located towards the bottom right of the MedSavant window. Each user must be given a name and password, and assigned a group (). MedSavant currently supports three types of groups: Administrators, Users, and Guests. Only administrators can upload new variants, create new patients, or access the Admin section to edit users and projects. Both users and administrators are allowed to define region sets (gene lists), whereas guests are not able to write any changes (i.e. they have read only access).
