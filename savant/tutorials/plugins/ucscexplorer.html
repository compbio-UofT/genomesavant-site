UCSC Explorer Plugin

The UCSC Explorer Plugin is a datasource plugin that permits the loading tracks from UCSC's database in much the same way as in the UCSC Genome Browser.

Install the UCSC Explorer Plugin via the Savant Plugin Repository. Restart Savant and open Human hg18 genome and RefSeq genes.

We'll now open the UCSC Explorer plugin via the tab at the bottom of the screen. The UCSC Explorer resembles the UCSC Table Browser. Switch to hg18 - Human - Mar. 2006 (NCBI36/hg18). Expand the Comparative Genomics section. Click the phyloP46wayAll - Vertebrate Cons checkbox, the click the Load Selected Tracks button. This track contains sequence conservation across 46 vertebrates.
