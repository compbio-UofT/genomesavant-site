Part 4: Experimenting with plugins

Savant is rather unique in in its ability to incorporate plugins. Plugins can fall into many categories:

    Datasource: enable access to an external datasource (e.g. UCSC and 1000 Genomes)

    Analytics: perform custom analytics in realtime (e.g. Cufflinks)

    Visualization: display custom graphics on top of tracks (e.g. SNP Finder)

    More (e.g. Table View)

Installation

Plugins can be installed directly from within Savant. To do so, open the Plugin Manager by choosing Plugins > Plugin Manager > Install from Repository. The repository contains an updated list of plugins that are available for various versions of Savant.

Here is a list of plugins that work with the version of Savant used in this lab:

    Chromatogram - Shows Sanger sequencing chromatograms overlaid on the reference genome

    edgeR - Detects and visualizes differential enrichment from RNA-seq or ChIP-seq data

    Data table - Shows textual data from track records in tabular form

    GATK - Predicts and visualizes genotypes from read alignment tracks

    Gene ontology - Guides navigation based on ontology terms

    PING - Guides navigation based on proteinâ€“protein interaction databases

    Remote commander - Issue navigation and other commands remotely through external tools

    Ribosome - Shows the amino acid translation of gene tracks

    RNA-seq analyzer - Reconstructs and estimates isoforms from RNA-seq data

    SimpleSNP - Predicts and visualizes SNVs from read alignment tracks

    Snapshot - Exports track images at every bookmarked genomic region

    SRMA - Realigns HTS read alignments

    UCSC explorer - Provides a graphical interface for loading UCSC tracks

    WikiPathways - Guides navigation based on biological pathways

Experiment with plugins of interest. A simple exercise with the UCSC plugin is described below.

UCSC Explorer Plugin

The UCSC Explorer Plugin is a datasource plugin that permits the loading tracks from UCSC's database in much the same way as in the UCSC Genome Browser.

Install the UCSC Explorer Plugin via the Savant Plugin Repository. Restart Savant and open Human hg18 genome and RefSeq genes.

We'll now open the UCSC Explorer plugin via the tab at the bottom of the screen. The UCSC Explorer resembles the [http://genome.ucsc.edu/cgi-bin/hgTables?command=start UCSC Table Browser]. Switch to hg18 - Human - Mar. 2006 (NCBI36/hg18). Expand the Comparative Genomics section. Click the phyloP46wayAll - Vertebrate Cons checkbox, the click the Load Selected Tracks button. This track contains sequence conservation across 46 vertebrates.
