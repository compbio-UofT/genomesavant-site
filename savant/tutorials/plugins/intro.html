Overview

After this lab, you will be able to:

  • browse and install plugins
  • use plugins to perform realtime analytics, view new types of visualizations, and connect to external datasources

Requirements

It is recommended that you go through this tutorial with the latest version of Savant. To check if you have the latest version, open Savant and click Help > Check for updates.

Troubleshooting

If you are performing this lab during a workshop, feel free to ask the instructor for help. Otherwise, please email us at support@genomesavant.com.
